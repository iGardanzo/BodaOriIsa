# Wedding Invite (Local)

## Run locally (no build tools)
Option 1 (Python):
- Open a terminal in this folder and run:
  python -m http.server 8000
- Then open:
  http://localhost:8000

Option 2 (Node):
- npx serve
- open the URL shown.
